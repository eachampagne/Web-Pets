### Getting Started

To get started:

```
npm install
npm run build
npm run style
npm start
```

When deploying to a new location, remember to create and populate the .env file and to register the app with the Google Cloud Platform to ensure authentication works properly.

### Tech Stack

* Client-side: React with Hooks
* Server-side: Node.js and Express
* Database: MongoDB with Mongoose
* Authentication: Passport with Google Authentication
* Styling: Tailwind CSS
* Building: Webpack and Babel
* External API: [Weather API](https://www.weatherapi.com/)
